# Analog IC Design Portfolio

This repository contains analog and mixed-signal circuit design projects implemented and simulated using SPICE-based tools.

The goal of this portfolio is to demonstrate practical understanding of analog circuit design concepts such as dynamic comparators, switched-capacitor circuits, and data converter architectures.

## Projects

### 1. Switched Capacitor DAC

Design and simulation of a switched capacitor digital-to-analog converter using MOS switches and reference voltages.

Features:

* Charge redistribution DAC architecture
* Clock controlled switching
* RC output filtering
* Transient simulation analysis

### 2. StrongARM Dynamic Comparator

Implementation of a high-speed dynamic latch comparator commonly used in SAR and Flash ADC architectures.

Features:

* Differential input pair
* Cross coupled regenerative latch
* Clock controlled operation
* Transient comparison behavior

## Tools Used

* SPICE simulation environment
* Analog circuit design principles
* Transient analysis for waveform verification
